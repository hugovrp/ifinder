# Interface Web - IFinder

Frontend desenvolvido para interação com o agente de IA IFinder.

A interface foi projetada e prototipada utilizando a plataforma **AntiGravity**, com o objetivo de conhecer a plataforma e suas capacidades, acelerando o ciclo de desenvolvimento e proporcionando uma boa experiência do usuário (UX) de forma ágil.

### Detalhes Técnicos

A aplicação é uma SPA (Single Page Application) construída com as tecnologias:

*   **HTML5**
*   **CSS3**
*   **JavaScript**

### Funcionalidades

*   **Gerenciamento de Sessões**: Uso de `localStorage`, permitindo retomada rápida de sessões anteriores.
*   **Interface Reativa**: Feedback visual imediato para ações do usuário (Loading states, Toasts de notificação).
*   **Integração com Backend**: Consome endpoints RESTful (`/chat`, `/sessions`, `/auth`) para orquestração das interações com o agente de IA.
*   **Layout Responsivo**: A interface é projetada para funcionar em dispositivos móveis e desktops.
